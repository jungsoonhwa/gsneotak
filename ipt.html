IT Business : IP Contact Center

  • Cisco Unified Contact Center Enterprise(이전의 Cisco IP Contact Center [IPCC] Enterprise)는 지능형 연락 라우팅, 통화 처리, 네트워크 - 데스크탑 사이의 CTI(Computer Telephony Integration), IP 인프라를 통한 멀티채널 연락 관리 기능 등을 제공합니다. Cisco Unified Contact Center Enterprise는 멀티채널 ACD(Automatic Call Distributor)기능과 IP 텔레포니를 단일 솔루션으로 연결하므로 기업은 분산된 연락 센터 인프라를 신속하게 배포할 수 있습니다

사업개요

  • IP 텔레포니(Telephony)는 기존 TDM 방식의 기업 통신망을 데이터 네트워크를 이용한 인터넷 프로토콜(IP)방식으로 기업의 통신 환경에 혁신을 가져오고 있습니다.

IP Telephony Solution

  • 기존 기업 통신 환경에서 운영되던 음성, 데이터 통신을 하나의 네트워크로 통합 운영할 수 있는 IP Telephony Solution (IPT)과 IP네트워크를 통해 고객과 상담원을 연결하는 지능형 콜센터 솔루션인 IP Telephony Contact Center Solution (IPCC)을 제공합니다. 신한은행, 메리츠화재 등의 금융기업, 현대택배 등 일반 대기업, 공공기관 등에 서비스를 제공하여 고객의 Voice통신 환경의 편의성을 극대화 합니다.

IP Contact Center

  • Cisco Unified Contact Center Enterprise(이전의 Cisco IP Contact Center [IPCC] Enterprise)는 지능형 연락 라우팅, 통화 처리, 네트워크 - 데스크탑 사이의 CTI(Computer Telephony Integration), IP 인프라를 통한 멀티채널 연락 관리 기능 등을 제공합니다. Cisco Unified Contact Center Enterprise는 멀티채널 ACD(Automatic Call Distributor)기능과 IP 텔레포니를 단일 솔루션으로 연결하므로 기업은 분산된 연락 센터 인프라를 신속하게 배포할 수 있습니다.

IPCC솔루션 구성

  • 시스코 IPCC 솔루션은 PSTN 연동과 IP Phone, Voice Gateway 관리를 위한 Call Manager와 Call Queing, 사용자 정보 입력을 위한 IP IVR, Skill Based 라우팅 Screen Pop up을 위한 ICM으로 구성되어 있습니다.
